# hospital-server-api
医院服务器后端api


技术选型：
后端技术栈
框架: Spring Boot 2.7+
数据库: MySQL 8.0 + MyBatis Plus
缓存: Redis 6.0+
消息: WebSocket + STOMP
认证: JWT + Spring Security
监控: Spring Boot Actuator + Micrometer
前端技术栈
框架: Vue 3 + Composition API
UI库: Element Plus
状态管理: Pinia
路由: Vue Router 4
HTTP: Axios
WebSocket: STOMP.js
构建: Vite
